#### 项目目录结构划分
- src
	- api （抽取出API请求）
	- assets （静态资源）
	- common （公共js）
	- components （公共组件）
	- router （跳转路由）
	- store （状态管理）
	- views （视图，如首页视图、分类视图）
	- App.vue （根组件）
	- main.js （入口js）
- .editorconfig （ESLint配置，代码风格）
- .gitignore （github配置）
- postcss.config.js （移动端适配单位转换 postcss-px-to-viewport）
- vue.config.js （vue-cli配置）
